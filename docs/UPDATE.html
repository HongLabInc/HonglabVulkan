PBR 셰이딩 및 카메라 수정 (2026-04-07)

  • 월드 카메라 위치 계산 수정 — SceneUBO::cameraPos에 view 행렬 생성용 내부 파라미터(camera_.position)를 그대로 넘기고 있어, 셰이더의 view 벡터 V = normalize(cameraPos - fragPos)가 실제 카메라와 어긋나는 문제를 수정했습니다. inverse(view)[3]으로 월드 카메라 위치를 추출하도록 변경.
  • IBL specular reflect 부호 수정 — getIBLRadianceGGX()에서 reflect(v, n)으로 호출하던 것을 reflect(-v, n)으로 수정했습니다. GLSL reflect()는 incident vector(camera→surface)를 받는데, v는 surface→camera 방향이므로 부호를 뒤집어야 합니다. 이 브랜치는 deferred 렌더링을 사용하므로 실제 사용되는 셰이더는 deferredLighting.comp이며, pbrForward.frag(미사용)도 일관성을 위해 함께 수정했습니다. Sascha Willems pbribl.frag의 패턴과 동일. 참고로 Khronos Vulkan-Samples의 shaders/shader_object/material_reflective.frag에도 동일한 부호 오류가 남아 있는 것을 확인했습니다.
  • Metallic / Roughness 슬라이더 라벨 수정 — UI 슬라이더 라벨이 셰이더의 coeffs[4]=metallic, coeffs[5]=roughness 매핑과 뒤바뀌어 있던 문제를 수정했습니다.

Tracy 0.13 업그레이드 및 G-buffer 수정 (2026-03-31)

  • Tracy 프로파일러를 0.11.1에서 0.13.1로 업그레이드했습니다.
  • 0.13에서 __LINE__ constexpr 문제가 수정되어 TRACY_NO_CALLSTACK 패치가 더 이상 필요 없습니다.

vcpkg 설정

  • vcpkg manifest 모드 사용 (vcpkg.json)
  • Directory.Build.props 추가 — Tracy 0.13의 변경된 헤더 경로를 솔루션 전체에 적용

알려진 이슈: Tracy debug lib 경로 버그

  • Tracy 0.13의 CMake install이 debug lib을 debug/lib/Debug/에 설치하지만, vcpkg auto-link는 debug/lib/만 탐색합니다.
  • 원인: wolfpld/tracy#1124에서 config별 하위 디렉토리 설치가 추가됨. vcpkg는 이미 debug/release를 별도 prefix로 분리하므로 이중 분리가 발생.
  • 증상: Debug 빌드에서 LNK2019: unresolved external symbol tracy::... 링크 에러
  • 임시 해결: vcpkg_installed 설치 후 수동 복사 필요 
    copy vcpkg_installed\x64-windows\x64-windows\debug\lib\Debug\TracyClient.lib vcpkg_installed\x64-windows\x64-windows\debug\lib\
copy vcpkg_installed\x64-windows\x64-windows\debug\bin\Debug\TracyClient.dll vcpkg_installed\x64-windows\x64-windows\debug\bin\
  • vcpkg Tracy 포트가 수정되면 이 단계는 불필요해집니다.

G-buffer layout transition 수정

  • G-buffer color attachment(gAlbedo, gNormal, gPosition, gMaterial)에 명시적 layout transition이 빠져있었습니다.
  • pbrDeferred 렌더 후 deferredLighting compute shader가 이미지를 읽을 때 VUID-vkCmdDispatch-imageLayout-00344 validation error가 발생.
  • Renderer.cpp에서 color attachment 사용 전 transitionToColorAttachment(cmd) 호출 추가.
